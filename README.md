
# Sobre este repositorio:

Feed RSS alterno de crunchyroll
## 🛜Uso
Agregar el link del RSS resultante en la carpeta docs a su lector de preferencia, se puede usar [esta URL](https://raw.githubusercontent.com/Danieruhuli/simulcast-rss/main/docs/simulcast-rss.xml)



## 🖥️ Razón
Existe un feed oficial RSS de Crunchyroll el cual es: https://www.crunchyroll.com/rss?lang=esLA sin embargo en el 2025 fue dado de baja y trás solicitud de varios usuarios, volvieron a activarlo. Pero meses después de ello, el feed ha estado fallando y no muestra toda la información como antes. Por esta razón decidí crear un feed alterno que mostrara los programas agregados, aunque no logré que fuera tan frecuente ni especifico como el oficial.

## 📖 MangaPlus
Se Agrega RSS de MangaPlus ya que la página no posee uno, se tienen 2 opciones
1. Actualización de cada capitulo nuevo que salga usando [esta URL](https://raw.githubusercontent.com/Danieruhuli/simulcast-rss/main/docs/mangaplus-rss.xml)
2. Actualización de solo One-Shots y Obras nuevas (se considera nueva si tiene máximo 4 capitulos) usando [esta URL](https://raw.githubusercontent.com/Danieruhuli/simulcast-rss/main/docs/mangaplus-new-rss.xml)
